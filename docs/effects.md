# Эффекты применяются для страницы и для элементов.

## Эффекты для страницы

Для применения эффектов для страницы установите атрибуты `data-in-effect` (появление) и `data-out-effect` (исчезновение) в одно из следующий значений.

|            название                   |                       описание                    |
|:--------------------------------------|:---------------------------------------------------|
| `appear-from-top`                     | появление сверху								   |
| `appear-from-bottom`					| появление снизу							     	   |
| `appear-from-transparency`			| проявление через прозрачность				     |
| `appear-from-top-and-transparency`	| появление сверху и проявление через прозрачность |
| `appear-from-bottom-and-transparency`	| появление снизу и проявление через прозрачность  |
| `disapear-to-top`						| смена съездом вверх							   |
| `disapear-to-bottom`					| смена съездом вниз							   |
| `disapear-become-transparent`			| исчезновение через прозрачность				   |

Рекомендуемые значения: `data-in-effect="appear-from-bottom"` `data-out-effect="disapear-to-top"`

## Эффекты для элементов

Внутри каждой страницы вы можете добавлять новые эффекты, которые будут применяться при перелистывании (`scroll` / `swipe`).

Чтобы добавить эффект используйте конструкцию

```html
  <div class="animation-element"
       data-animation-step="1"
	   data-inactive-class="anim-invisible"
	   data-active-class="anim-bounce-from-bottom"
	   >
	   содержимое элемента
  </div>	   
```

- Указание класса `animation-element` обязательно.
- Значение содержимого элемента может быть любым.
- `data-animation-step` - номер анимации внутри страницы - простое число
- `data-inactive-class` - класс, описывающий состояние элемента до применения анимации (`anim-invisible` - невидимый)
- `data-active-class` - класс, описывающий состояние элемента во время анимации

После добавления эффектов на странице необходимо добавить ей атрибут `data-steps-total` со значением, равным количеству элементов анимации внутри страницы + 1.

Например, для страницы, содержащей 3 элемента анимации атрибут будет записываться как `data-steps-total="4"`.

### Состояния элементов во время анимации

|          название         |            описание             |
|:--------------------------|:---------------------------------|
| `anim-bounce-from-bottom` | появление снизу с отскоком      |
| `anim-bounce-from-top`    | появление сверху с отскоком     |
| `anim-appear-from-left`   | появление слева                 |
| `anim-appear-from-right`  | появление справа                |
